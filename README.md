
# NutriScan

Escaneo de tablas nutricionales


## Ejecutar localmente

Clonar

```bash
  git clone https://github.com/Lespinald/nutriscanapp_front
```






ir al repositiorio

```bash
  cd {carpeta_proyecto}
```

Instalar dependencias

```bash
  npm install
  npm i -g serve
```

Iniciar ambiente de desarrollo

```bash
  npm run dev
```

Iniciar ambiente de produccion localmente

```bash
  npm run build
  serve dist
```
